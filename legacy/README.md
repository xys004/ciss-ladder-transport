# Legacy Scripts

This directory preserves the original research scripts with minimal intervention.

## Why keep them here

- to preserve the exact structure used during the research workflow,
- to make the Zenodo archive historically traceable,
- to avoid silently changing numerical behavior before publication.

## Important caveats

- These scripts execute work at top level.
- Two post-processing scripts are tied to Google Colab and Google Drive.
- Some names were normalized only at the file level; the code bodies were not cleaned up here.
- For archival integrity, the preserved files should be treated as reference implementations.
